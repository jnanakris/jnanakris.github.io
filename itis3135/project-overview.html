Project Overview

Description & Purpose

Travel Companion Match is a web application that connects air travelers in need of assistance— such as seniors, minors, or solo passengers—with volunteer co-passengers on the same flight. Travelers submit flight details anonymously, and matched volunteers are notified using virtual IDs for secure communication. The system facilitates meaningful, trust-based connections to help those who need travel assistance.

Intended Users
